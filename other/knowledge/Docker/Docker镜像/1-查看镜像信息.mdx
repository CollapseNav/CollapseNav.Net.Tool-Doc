---
title: 查看镜像信息
sidebar_position: 1
---

Docker提供了一个非常好用的 `inspect` 命令来查看镜像的详细信息, 与之前提到的 `docker images` 命令不一样, 那个只能列出有什么镜像, 只能看到一些非常简单的表面的信息, 而 `inspect` 命令可以查看包括镜像的ID, 标签, 父镜像, 容器的配置, 卷, 环境变量等更加详细的信息。

```bash
docker inspect nginx:alpine
```
