## 创建项目

初始化项目是个非常简单的事情

```shell
yarn create docusaurus
```

如果是npm, 则使用如下命令

```shell
npm init docusaurus
```

初始化时会让你设置文档项目的名称, 一般来说随便起个名字就好, 最后会用这个名字创建一个新的文件夹, 项目的文件都放在这个文件夹里
