## 类图

```text
    ```mermaid
    classDiagram

    class Person{
        +SayHelloWorld(HelloWorld)
    }
    class HelloWorld{
        +World World
        -string Greetings
        +HelloWorld()
    }
    class World{
        +string WorldName
    }

    Person ..> HelloWorld
    HelloWorld ..> World
    ```
```


```mermaid
classDiagram

    +SayHelloWorld(HelloWorld)
}
class HelloWorld{
    +World World
    -string Greetings

}
class World{
    +string WorldName
}

```

## 流程图

```txt
    ```mermaid
    flowchart TD
        start([开始])
        finish([结束])
        func1[操作1]
        cond1{判断1}
        savedata[保存]
        rollback[回退]
        db[(数据库)]
        savedata-->db
        rollback-->db
        start-->func1-->cond1-->|yes|savedata-->finish
        cond1-->|no|rollback-->finish
    ```
