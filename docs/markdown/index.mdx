# Markdown环境配置

## 安装 vscode

强烈推荐使用 [vscode](https://code.visualstudio.com/)

开源免费跨平台, 而且有很多插件可以用, 体验很棒

如果直接下载速度太慢, 可以把地址下载地址换成国内的cdn

复制原来的下载地址, 将域名部分替换为`https://vscode.cdn.azure.cn`即可

## 安装markdown插件

虽然原始状态下的vscode也可以预览markdown文件, 但装了插件会更好用一些

使用快捷键 `ctrl` + `shift` + `x` 打开插件页, 然后搜索以下插件并安装(也可以直接点击插件名字跳转安装)

1. [**Markdown All in One**](vscode:extension/yzhang.markdown-all-in-one)
2. [**Markdown Preview Enhanced**](vscode:extension/shd101wyy.markdown-preview-enhanced) (可以获得更加酷炫的预览效果)
3. [**markdownlint**](vscode:extension/DavidAnson.vscode-markdownlint) (主要是用来语法提示之类的,可装可不装)
4. [**Markdown Preview Mermaid Support**](vscode:extension/bierner.markdown-mermaid) (拓展画图功能)
6. [**Markdown Extended**](vscode:extension/jebbs.markdown-extended) (虽然不记得有啥用了，但是装就好了)

## Markdown文件

Markdown 文件的拓展后缀为 `.md`
直接新建一个文件名为 `XXX.md` 的文件即可开始写作！
