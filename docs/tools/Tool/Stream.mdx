# Stream

### ToBytes

:::note
流转为 bytes
:::

```csharp
var filePath = "./XXXXX";
using var fs = new FileStream(filePath, FileMode.OpenOrCreate);
var bytes = fs.ToBytes();
```

### ToStream

:::note
bytes 转为流
:::

```csharp
var filePath = "./XXXXX";
using var fs = new FileStream(filePath, FileMode.OpenOrCreate);
var stream = fs.ToBytes().ToStream();
```

### SaveTo

:::note
保存为文件
:::

```csharp
var filePath = "./XXXXX";
using var fs = new FileStream(filePath, FileMode.OpenOrCreate);
string toFile = "./PPPPP";
fs.SaveTo(toFile);
fs.ToBytes().SaveTo(toFile);
```

### Seek

:::note
流指针移动
:::

```csharp
var filePath = "./XXXXX";
using var fs = new FileStream(filePath, FileMode.OpenOrCreate);
fs.SeekToEnd();
fs.SeekToOrigin();
fs.SeekTo(0);
```

### Clear

:::note
清空流
:::

```csharp
stream.Clear();
```
