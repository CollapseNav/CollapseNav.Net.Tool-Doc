## 平台

<Tabs>
    <TabItem value='x99' label='x99'>
    
    :::tip
    最低价格

    | 主板    | cpu       | 价格          |
    | ------- | --------- | ------------- |
    | 科脑x99 | e5-2620v4 | 138 + 8 = 146 |
    :::

    x99指的主要是 `e5v3` 和 `e5v4` 这两种服务器cpu，两者之间没有什么本质的区别，都可以用 

    **优点**

    * 可以多核心
    * 可以多cpu
    * 可扩展性强
    * 价格便宜
    * 可以使用便宜的ECC服务器内存

    **缺点**
    
    * 老平台
    * 多核心的cpu主频较低
    * 高主频的cpu价格较高
    * 高功耗

    </TabItem>
    <TabItem value='e3v3' label='e3v3'>

    :::tip
    最低价格

    | 主板 | cpu       | 价格          |
    | ---- | --------- | ------------- |
    | b85  | e3-1231v3 | 60 + 90 = 150 |
    :::

    **优点**

    * 价格便宜
    * 主频较高
    * 可使用桌面端主板
    * 性能接近甚至超过同代i7

    **缺点**

    * 老平台
    * 集显版本较贵
    * 可能需要搭配独显使用

    </TabItem>
    <TabItem value='e3v5/6' label='e3v5/6'>

    :::tip
    最低价格

    | 主板     | cpu       | 价格           |
    | -------- | --------- | -------------- |
    | 魔改H110 | e3-1220v5 | 120 + 15 = 135 |
    :::

    个人感觉 `e3v5` 和 `e3v6` 没啥差别，可以使用相同主板

    **优点**

    * 价格便宜
    * 主频较高
    * 制程相较 `e3v3` 新一点

    **缺点**

    * 老平台
    * 集显版本较贵
    * 可能需要搭配独显使用
    * 不能直接使用桌面端主板，需要魔改bios

    </TabItem>
